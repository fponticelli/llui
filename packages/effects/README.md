# @llui/effects

Effect builders for [LLui](https://github.com/fponticelli/llui).

http, cancel, debounce, sequence, race, storage, broadcast + `Async<T, E>` type, `ApiError` union, `handleEffects().use()` chain, and `resolveEffects()` for SSR data loading.

```bash
pnpm add @llui/effects
```
